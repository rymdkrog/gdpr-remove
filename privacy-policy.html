Privacy Policy — GDPR Remove

Last updated: 2026-03-30

What data does this extension collect?

This extension stores the following data locally in your browser:

  • Enabled/disabled toggle — whether the extension is active.
  • Request counter — how many GDPR emails have been triggered.

This data never leaves your browser. It is stored using Chrome's local storage API and is only accessible to this extension.

What does this extension do?

When you close a browser tab showing aftonbladet.se, the extension opens a pre-filled mailto: link in your default email client. You must manually review and send the email yourself.

Does this extension track my browsing?

No. The extension checks tab URLs only to detect aftonbladet.se pages. It does not log, store, or transmit any browsing history or URLs.

Does this extension send data to any server?

No. The extension has no backend, no analytics, and makes no network requests. All functionality runs entirely within your browser.

How can I delete my data?

Click the "Nollställ" (reset) button in the extension popup to clear the counter, or remove the extension entirely from chrome://extensions.
